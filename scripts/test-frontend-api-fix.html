🔍 问题分析

根本原因：

  • useReservation.js 使用 useApi() 的 request 方法
  • useApi.js 中硬编码了 baseURL = 'http://localhost:3000'
  • 实际服务器地址不是 localhost:3000
  • 导致前端请求发送到错误的地址

修复方案：

  • 在 useApi.js 中添加动态 baseURL 配置
  • 使用环境检测自动选择正确的服务器地址
  • 统一 API 配置管理
