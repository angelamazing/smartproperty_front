测试修复后的初始加载和按钮点击加载功能

问题描述

原始问题

错误信息: reservation.js? [sm]:1 加载预约记录失败: Error: 获取场地详情失败

现象: 第一次进入球馆预约界面时加载失败，但点击全部、羽毛球等按钮时正常工作

根本原因分析

问题根源

在 loadInitialData() 方法中，并行调用了三个方法：

  • this.loadVenues()
  • this.loadReservationRecords()
  • this.loadScheduleData()

但这些方法都没有返回值，导致：

  • venues 变量是 undefined
  • 被赋值给 this.venues，覆盖了原本在方法内部设置的值
  • 导致界面显示异常

修复方案

1. 修复 loadVenues 方法

async loadVenues() {
  try {
    // ... 原有逻辑 ...
    if (response && response.success) {
      const venues = response.data?.list || []
      
      // 为每个场地生成时间段
      for (const venue of venues) {
        venue.timeSlots = await this.generateTimeSlots(venue)
      }
      
      this.venues = venues
      return venues  // ✅ 添加返回值
    }
  } catch (error) {
    // ... 错误处理 ...
    this.venues = []
    return []  // ✅ 添加返回值
  }
}

2. 修复 loadReservationRecords 方法

async loadReservationRecords() {
  try {
    // ... 原有逻辑 ...
    if (response && response.success) {
      const records = response.data?.records || []
      // ... 处理逻辑 ...
      this.reservationRecords = records
      return this.reservationRecords  // ✅ 添加返回值
    }
  } catch (error) {
    // ... 错误处理 ...
    this.reservationRecords = []
    return []  // ✅ 添加返回值
  }
}

3. 修复 loadScheduleData 方法

async loadScheduleData() {
  try {
    // ... 原有逻辑 ...
    if (response && response.success) {
      const scheduleData = response.data || []
      this.scheduleData = scheduleData
      return scheduleData  // ✅ 添加返回值
    }
  } catch (error) {
    // ... 错误处理 ...
    this.scheduleData = []
    return []  // ✅ 添加返回值
  }
}

4. 改进 loadInitialData 方法

async loadInitialData() {
  try {
    const [venues, records, schedule] = await Promise.all([
      this.loadVenues(),
      this.loadReservationRecords(),
      this.loadScheduleData()
    ])

    // 确保数据正确设置（双重保险）
    if (venues) this.venues = venues
    if (records) this.reservationRecords = records
    if (schedule) this.scheduleData = schedule
    
    console.log('初始数据加载完成:', {
      venuesCount: this.venues?.length || 0,
      recordsCount: this.reservationRecords?.length || 0,
      scheduleCount: this.scheduleData?.length || 0
    })
  } catch (error) {
    console.error('加载初始数据失败:', error)
    uni.showToast({
      title: '数据加载失败',
      icon: 'none'
    })
  }
}

为什么点击按钮时正常？

按钮点击逻辑

点击"全部"、"羽毛球"等按钮时，直接调用 this.loadVenues() 方法：

selectVenueType(venueType) {
  this.selectedVenueType = venueType
  this.loadVenues()  // 直接调用，不依赖返回值
}

这个方法内部已经正确设置了 this.venues，所以界面能正常显示。
