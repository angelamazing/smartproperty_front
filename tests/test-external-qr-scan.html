外部扫码用餐确认功能测试

测试时间: 2025-09-13 | 功能: 外部用户扫码确认就餐

🎯 功能概述

外部扫码用餐确认功能允许外部用户通过扫描二维码快速确认就餐，系统会根据扫码时间自动判断餐次（早餐、午餐、晚餐），并提供完整的错误处理机制。

📱 二维码生成

支持生成带参数的二维码，包含scene_token等验证信息

  • 普通二维码生成
  • 微信小程序码生成
  • 带URL参数二维码
  • 安全令牌二维码

🔍 扫码处理

智能解析二维码内容，提取scene_token进行验证

  • URL参数解析
  • JSON格式解析
  • 直接token解析
  • 错误格式处理

⏰ 时间判断

根据扫码时间自动判断餐次，确保在正确时间内确认

  • 早餐时间判断
  • 午餐时间判断
  • 晚餐时间判断
  • 非就餐时间处理

🛡️ 安全验证

完整的验证机制确保扫码安全可靠

  • scene_token验证
  • 二维码过期检查
  • 重复确认检查
  • 用户身份验证

🔄 扫码流程

1. 生成二维码
→
2. 用户扫码
→
3. 解析参数
→
4. 验证token
→
5. 判断餐次
→
6. 确认就餐

⏰ 就餐时间规则

📅 餐次时间安排

🌅 早餐 06:00 - 10:00
☀️ 午餐 11:00 - 14:00
🌙 晚餐 17:00 - 20:00

注意：只有在规定时间内扫码才能成功确认就餐，其他时间会返回错误信息。

🔧 API接口

生成外部扫码二维码

接口：POST /api/qr-scan/generate-external-qr

参数：

{ "width": 430, "margin": 2, "sceneToken": "dining_1705123456789_abc123def", "expireTime": 3600, "mealType": "auto" }

生成微信外部扫码码

接口：POST /api/qr-scan/generate-wechat-external-code

参数：

{ "width": 430, "scene": "scene_token=dining_1705123456789_abc123def", "page": "pages/dining/external-scan", "expireTime": 3600 }

外部扫码确认就餐

接口：POST /api/qr-scan/external-scan

参数：

{ "sceneToken": "dining_1705123456789_abc123def", "scanTime": "2025-09-13T12:30:00.000Z", "userInfo": { "nickName": "用户昵称", "avatarUrl": "头像URL", "openId": "微信openId" } }
