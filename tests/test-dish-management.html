📊 接口对接说明

请求参数

菜品列表接口现在支持以下餐次类型相关参数：

{ "page": 1, "pageSize": 20, "keyword": "搜索关键词", "categoryId": "分类ID", "status": "active|inactive", "isRecommended": "true|false", "mealType": "breakfast|lunch|dinner" }

菜品数据现在包含餐次类型信息：

{ "_id": "dish-001", "name": "小笼包", "meal_types": ["breakfast"], "isRecommended": 1, "status": "active", // ... 其他字段 }

🚀 使用说明

管理员操作流程

  1. 进入菜品管理页面
  2. 使用筛选器或快速筛选按钮按餐次类型筛选菜品
  3. 点击"新建菜品"创建新菜品，选择适用的餐次类型
  4. 编辑现有菜品时，可以修改餐次类型设置
  5. 菜品列表会显示每个菜品适用的餐次类型标签
